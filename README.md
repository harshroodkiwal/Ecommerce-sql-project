E-Commerce Sales Analysis (SQL Project)

# Overview
This project analyzes e-commerce sales data using SQL to extract meaningful business insights.

# Objectives
- Analyze monthly sales trends  
- Identify top customers  
- Find repeat customers  
- Analyze city-wise sales  
- Calculate average order value  

# Tools Used
- SQL  

# Dataset
- customers.csv → customer details  
- orders.csv → order transactions  

# Project Files
- queries.sql → SQL queries  

# Key Insights
- Identified top customers based on spending  
- Analyzed monthly sales trends  
- Found repeat customers contributing to revenue
