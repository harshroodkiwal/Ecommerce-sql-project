# E-Commerce Sales Analysis (SQL Project)

# Overview
This project analyzes e-commerce sales data using SQL to extract meaningful business insights such as customer behavior, sales trends, and key performance metrics.


# Objectives
- Analyze monthly sales trends  
- Identify top customers  
- Find repeat customers  
- Perform city-wise sales analysis  
- Calculate Average Order Value (AOV)  


# Tools Used
- SQL (MySQL)


# Dataset
The dataset includes:
- customers.csv → customer details (ID, name, city)  
- orders.csv → order transactions (date, amount)  


# Project Files
- queries.sql → SQL queries used for analysis  


# Key Analysis

### 1. Monthly Sales Trend
Tracks how sales change over time.

### 2. Top Customers
Identifies highest revenue-generating customers.

### 3. Repeat Customers
Finds customers with multiple purchases.

### 4. City-wise Sales
Analyzes regional performance.

### 5. Average Order Value
Measures average spending per order.


# Key Insights
- A few customers contribute most revenue  
- Sales show growth over months  
- Repeat customers are important for business  


# Business Impact
This project demonstrates how SQL can be used to support business decision-making by analyzing customer behavior and sales performance.
